## ðŸ§  How It Works
1. User uploads a resume (PDF).
2. The resume is converted into an image using PDF.js.
3. Files are stored using **Puter File System (FS)**.
4. Resume data and metadata are stored using **Puter KV storage**.
5. **Puter AI** analyzes the resume based on the provided job role and description.
6. AI-generated insights and scores are displayed to the user.
